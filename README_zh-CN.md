# 心界

`version: beta1.0`

[英文版](./README.md)

## 介绍

「心界」是一个面向创作团队的在线沟通、协作、分享的开源系统。目前完成基础架构和简单功能，后续会逐步完善。

## 环境

- php 7.4+
- mysql 5.7+
- redis 5.0+
- nodejs 18+

## 特性

- 基于 vue3 + thinkphp6 + arcodesign 框架开发
- 统一认证中心（JWT）

## 体验

- 演示地址：http://www.coresite.cn
